#Nashville Housing Data Exploration (SQL)

##Project Overview:
In this project, I explored a global Covid-19 dataset using SQL to analyze infection trends, what were the death rates and vaccination rates among different countries at different times.  
The main goal of this project was to extract meaningful insights from the raw dataset using useful SQL queries.

#Tools Used:
SQL
SQL Server 

#Dataset:
Covid-19 dataset containing useful information about cases, deaths, population, and vaccination across different countries and continents. 

#Key Questions Explored:

What is the percentage of deaths compared to total cases?
Which countries have the highest infection rate relative to their population?
Which countries recorded the highest number of total deaths?
How did COVID-19 cases evolve globally?
What percentage of the population has been vaccinated?


#SQL Techniques Used:

JOIN Operations,
Aggregate Functions (sum, max, count),
GROUP BY,
ORDER BY,
Window Functions,
CTEs,
Calculating Percentages. 


#Key Insights 
We can see from the results of the queries that some countries experienced significantly higher infection counts compared to their population.
Death rates varied among different countries and continents.
As the vaccination rollout progressed, death counts and infection rates decreased in many regions.
